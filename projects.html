This is where I showcase the projects I've built as part of my school assignments or personal learning. Each project is a new lesson!

1. Simple GUI design applied. (Python.Tkinter)

The Goal

Creating this simple GUI calculator was a fundamental exercise in learning Python's capabilities beyond the terminal. The primary goal was to gain hands-on experience with a standard Python GUI library, focusing specifically on widget placement, event handling, and binding functions to button clicks to implement the application's core arithmetic logic. This project solidified my understanding of how to translate backend code into a practical, interactive user interface.

What I Did

When creating the calculator, I first set up the GUI window using a Python library called Tkinter and designed the layout by arranging the display field and all numeric and operation buttons into a grid. I then implemented the core logic by writing event handler functions to capture button clicks, manage the input string, and use Python's built-in evaluation capabilities to calculate the final result, including basic error handling.

Key Takeaway

The main lesson was learning how to connect Python's code to the buttons and screen of a visible program. This taught me how to handle user clicks and show the results, making the code work as an actual, interactive application.
